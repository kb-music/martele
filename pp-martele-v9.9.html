Practice Player — Help & Instructions

Practice Player is a focused music-study player built for efficient instrumental practice. It combines audio playback, score viewing, waveform navigation, loop points, speed control, and track-version switching in one clear workspace.

It is especially useful for repetition-based work: you can jump inside the waveform, repeat short passages, slow the music down, restart from a saved point, and switch between accompaniment or part versions without breaking concentration.

Hotkeys

  • SpacePlay / Pause
  • Left ArrowSeek backward
  • Right ArrowSeek forward
  • Up ArrowIncrease playback speed by 1%
  • Down ArrowDecrease playback speed by 1%
  • RReset speed to 100%
  • SRestart from saved start point / stop to saved start
  • ASet loop start point
  • BSet loop end point
  • CClear loop
  • TTurn guided tooltips off / on
  • IShow / hide this Help panel
Tooltips are now gentle “first-time only” tips: each control shows its hint once, then stops bothering you. Press T to turn the guided tips off or back on for the remaining unseen controls.
On touch devices, use this button to change tooltip mode — no keyboard needed.
